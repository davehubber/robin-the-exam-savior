# Dark Assassin Animation Pack
This asset pack includes 12 high-quality 2D animations for your platformer, roguelike, or RPG game.

## Features:
- 12 unique animations (Idle, Walk, Run, etc.).
- Spine-compatible files: .json, .atlas, .png.
- Individual PNG frames for custom setups.

## How to Use:
1. Import the `.json` and `.atlas` files into Spine or your game engine.
2. If needed, use PNG frames from the `Frames` folder for manual setups.
